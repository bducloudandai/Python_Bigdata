# LDA 모델 생성 및 훈련

# gensim 패키지의 LDA 모듈을 이용하여 토픽 모델 객체인 lda_model을 생성
k = 4  #토픽의 개수 설정
lda_model = gensim.models.ldamulticore.LdaMulticore(corpus, iterations = 12, num_topics = k, id2word = dictionary, passes = 1, workers = 10)
  • gensim.models.ldamulticore.LdaMulticore 클래스는 병렬로 처리할 수 있는 LDA 모델로, 학습 속도를 향상시켜주기 위해 복수의 CPU 코어를 사용
  • corpus는 학습 데이터를 나타내며, 각 문서에 대한 단어 빈도수 벡터를 포함
  • iterations=12는 각 업데이트 단계에서의 예상 최적화 반복 횟수입니다.
  • num_topics=k는 확인하려는 토픽의 개수
  • id2word=dictionary는 단어와 단어 ID간의 매핑을 나타내는 사전 객체
  • passes=1은 전체 학습 데이터에 대한 학습 반복 횟수
  • workers=10은 동시에 실행되는 작업 스레드의 수
